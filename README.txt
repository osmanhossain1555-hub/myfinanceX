Finance X PWA

Files:
- index.html
- manifest.webmanifest
- service-worker.js
- icon.svg

How to install:
1. Host this folder on HTTPS or open through localhost.
2. Open index.html in Chrome or Edge.
3. Use the Install App button or browser install prompt.

Quick local test:
- Python: python3 -m http.server 8000
- Then open http://localhost:8000/finance-x-pwa/

GitHub Pages:
- Upload these files to a repo root
- Enable GitHub Pages
- Open the site and install
